# Decisions

Use this folder for durable project decisions that future agents should consider.

Do not use it for daily logs or handoff notes.

## Decision Template

```md
# Decision: Title

Date:
Status: active | superseded | uncertain

## Decision

What was decided.

## Why

Why this is the current direction.

## Verification

How a future agent should confirm this still applies.

## Superseded By

Link to the newer decision when replaced.
```
